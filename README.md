**Summercart  -Summer eEssential Store**

SunCart is a modern summer-themed eCommerce web application built with **Next.js**, **Tailwind CSS**, **DaisyUI/HeroUI**, and **BetterAuth**.  
The platform allows users to explore seasonal summer products such as sunglasses, skincare items, summer outfits, beach accessories, and more. Users can browse products, view product details, register, login, and manage their profile after authentication.



**Project Purpose**

The main purpose of this project is to create a responsive and user-friendly summer eCommerce platform where users can explore summer essentials and access protected product details after logging in. The application also includes authentication, profile management, and user information update functionality using BetterAuth.
